1枚目
1枚目は多摩川河川敷。既にサイクリング中の方や河川敷で運動や娯楽を楽しんでいる人が多く、三密にならない程度に行動していました。
車両はE233系1000番台。現在、東京都近郊に行くとだいたいこの列車を見ることができ、モーター音やドアチャイムの種類も多くあります。
2枚目
2枚目は友人宅。今年の3月にデビューしたばかりのE261系サフィール踊り子は独特なデザインをしていて、横からみた青い車体はこの車両の目的地、伊豆の海を連想させます。
このサフィール踊り子は、今世界各国で流行っているウイルスの影響でデビューこそ静かなものでした。
しかし、いくら静かとはいえそんなに人が集まらなかったわけでもなく、東京駅にはそれなりに人は集まりました。
3枚目
3枚目は再び多摩川河川敷。1枚目と時間差はあまりなく、こっちの写真はE231系1000番台。東海道線、高崎線、宇都宮線、湘南新宿ラインなどで幅広く使用されている万能列車です。
しかし、2021年以降のE235系1000番台投入により、徐々に本数を減らしていきます。主要路線から完全に引退した後は、高崎方面の郊外地区で密かに活躍する予定です。
ですが、グリーン車、10両編成は廃止されてしまい、E231系とE233系の連結も見れなくなってしまうため、早めに記録しておくのがいいですね。
4枚目
4枚目は京急東神奈川駅。2100形ブルースカイトレインを撮影しました。実は2100形ブルースカイトレインはそんなに珍しくないです。
夢を壊すような事言ってしまって申し訳ないのですが、実は案外簡単な事で、2100形は基本三崎口~泉岳寺、もしくは泉岳寺~羽田空港のみでの運用なので、
京急蒲田~品川でみたらほぼ確定で数十分後には戻ってきます。逆に600形の方は当たり前のように成田空港まで行ってしまうので、日によりますが見れたらかなり運がいいと思っていいかもしれません。
